这个项目旨在解决 AI Agent 中 **Skills 功能重叠** 导致调用失效的问题。通过构建一个结构化的 Skills 管理系统，提升 Agent 的 Skill 选择准确率和调用成功率。

This project resolves **functional overlap** between AI Agent Skills that causes routing failures and incomplete capability use. It delivers a structured Skills management system that materially improves selection accuracy and invocation success rate.

**Headline numbers**
- 73 Skills fully classified in a functional matrix
- 8 medium-severity overlap pairs detected
- Routing accuracy **92%** (target 85%)
- Test coverage **100%** (8/8 passing)

## 🩺 核心问题

<table>
<tr>
<td width="33%" valign="top">

### 1️⃣ 功能重叠
多个 Skill 提供相似功能，Agent 不知道该选哪个。

**典型示例**
- `second-brain` ↔ `email-rag`
- `webex-transcript-download` ↔ `webex-space-dump`
- 多个 RADKit Skill 边界模糊

</td>
<td width="33%" valign="top">

### 2️⃣ 调用失效
Agent 选错 Skill 或不知道最佳使用场景，导致功能不完整、结果不准确。

**直接后果**
- 用户体验受损
- 重复尝试浪费 token
- 部分 Skill 长期"睡眠"

</td>
<td width="33%" valign="top">

### 3️⃣ 维护困难
新增 Skill 时难以判断是否与现有 Skill 重复，文档与实际能力脱节。

**长期影响**
- 库越来越乱
- 治理成本陡升
- 新成员上手难

</td>
</tr>
</table>

## ⚡ 快速开始

### 1. 查看核心文档

```bash
# RPI 研究 + 规划文档
cat rpi/skills-dedup-system/research.md
cat rpi/skills-dedup-system/plan.md

# 项目总结
cat PROJECT-SUMMARY-REPORT.md
```

### 2. 运行五个工具

```bash
# ⚡ 执行监控
python3 tools/skill_execution_monitor.py --help

# → 智能路由
python3 tools/skill_router.py --help

# ⌬ 重叠检测
python3 tools/overlap_detector.py --help

# ♥ 健康检查
python3 tools/health_check.py --help

# 📊 使用统计
python3 tools/usage_analyzer.py --help
```

### 3. 打开可视化仪表板

```bash
# macOS
open project-overview-dashboard.html

# 或在浏览器手动打开 skills-optimization/screenshots/ 下任意 .html
```
